# Simple Blog Application

My firts TypeScript project.  

I used following packages as well:  
- Tanstack Query  
- Axios  
- React Hook Form  
- React Router Dom  
- Json Server  
- Date fns  
And Tailwindcss for design  



To run the project

```npm
npm install  
npm run dev  
npx json-sever -p 3500 -w public/data/db.json

```
